# Stock-Termux
Make your termux clean from Bloatware. Install theme. A lightweight tool to remove all the bloatware from your terminal and make it attractive.

# Usage
1. git clone https://guthub.com/Saadkhan041/Stock-Termux
2. cd Stock-Termux
3. ls
4. python3 stock.py
